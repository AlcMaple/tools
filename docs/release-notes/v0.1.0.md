# 0.1.0 (2026-04-25)

MapleTools 首个公开发布版本。

> [!Note]
> 本应用依赖系统 ffmpeg，请确保已安装并加入 PATH。
> Windows 从 [ffmpeg.org](https://ffmpeg.org/download.html) 下载；macOS 推荐 `brew install ffmpeg`。

> [!Tip]
> macOS 首次打开提示"无法验证开发者"时，请右键 App → 选择"打开"，
> 或执行 `xattr -d com.apple.quarantine /Applications/MapleTools.app`。

> [!Warning]
> 当前为开发中版本（v0.x），界面与数据格式可能在后续版本调整。

### 新增

- 番剧搜索下载：支持 Xifan、Girigiri 两个站点的搜索与单集 / 全集下载
- 番剧资料：基于 Bangumi 抓取番剧详情（封面、简介、角色等）
- 本地媒体库：扫描指定目录的视频文件，自动提取缩略图
- 下载队列：单进程串行下载，支持暂停 / 恢复 / 重排
- 断点续传：Xifan 与 Girigiri 均支持暂停后续传，关闭应用后重启也能从原进度继续
- 搜索结果缓存：Xifan / Girigiri 30 天，BGM 14 天后台刷新
- 设置中心：下载路径、缓存开关、视频源选择

### 已知限制

- 未做代码签名 / 公证（macOS 首次打开需手动放行）
- 仅发布 Windows x64 与 macOS arm64
