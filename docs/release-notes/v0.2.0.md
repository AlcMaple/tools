围绕「追番」打造的一整套工作流上线，设置页重设计、文件浏览器增强。

> [!Note]
> 本应用依赖系统 ffmpeg，请确保已安装并加入 PATH。
> Windows 从 [ffmpeg.org](https://ffmpeg.org/download.html) 下载；macOS 推荐 `brew install ffmpeg`。

> [!Tip]
> macOS 首次打开提示"无法验证开发者"时，请右键 App → 选择"打开"，
> 或执行 `xattr -d com.apple.quarantine /Applications/MapleTools.app`。

> [!Warning]
> 当前为开发中版本（v0.x），界面与数据格式可能在后续版本调整。

---

### 新增

**追番系统**

- 新增「我的追番」汇总页，统一管理观看状态、当前集数、绑定的观看链接
- BGM 详情页和搜索卡片支持一键追番，一部番可同时绑定 Aowu / Xifan / Girigiri  / 自定义 URL 多个来源
- 我的追番列表行尾可一键在外部浏览器打开对应源的播放页继续观看
- 总集数支持手动填写，BGM 数据缺失（长寿番、特别篇、季番初期）也能继续跟进度
- 总集数留空时显示「连载中」

**番剧周历**

- 新增本季番剧周历页，按周一至周日列展示更新计划，当日列自动高亮

**云同步**

- 数据分开同步，互不影响
- 同步加冲突检测与确认弹窗，明确显示本地 / 远端差异避免误覆盖
- 首次升级会自动从老数据迁移追番列表

**设置页**

- 重新设计为左栏分类导航、自动保存
- 下载路径会显示当前实际生效的位置，留空时显示系统默认下载文件夹
- 一键打开当前下载路径

**文件浏览器**

- 占用中的文件自动杀进程后再删
- 右键菜单贴近屏幕边缘时自动翻转，光标始终贴在菜单角上
