BGM 搜索新增漫画 / 小说类目支持

> [!Tip]
> 已装 v0.3.0 的用户：应用会自动检测并提示，点「重启安装」即可。

> [!Note]
> 本应用依赖系统 ffmpeg，请确保已安装并加入 PATH。
> Windows 从 [ffmpeg.org](https://ffmpeg.org/download.html) 下载；macOS 推荐 `brew install ffmpeg`。

> [!Tip]
> macOS 首次打开提示"无法验证开发者"时，请右键 App → 选择"打开"，
> 或执行 `xattr -d com.apple.quarantine /Applications/MapleTools.app`。

> [!Warning]
> 当前为开发中版本（v0.x），界面与数据格式可能在后续版本调整。

---

### 新增

**BGM 漫画 / 小说支持**

- BGM 搜索可切换「动画」/「漫画小说」类目，按需选择不同维度搜索
- 「我的追番」从「追番列表 / 推荐」拆成「动画 / 漫画 / 小说 / 推荐」4 个平级 tab，老追番自动归入动画 tab

---

### 优化

- BGM 触发限流时显示倒计时刷新按钮，能看清还要等多久；强制点会有视觉警示
- 错误提示按具体原因分类（限流 / 超时 / 服务器异常 / BGM 偶发故障），不再统一显示"网络请求失败"
- 周历刷新失败时会明确反馈，不再"看似刷新成功但缓存没更新"
- 同步弹窗显示动画 / 漫画 / 小说三个维度计数，上传 / 拉取前能看清各类目数量

---

### 修复

- BGM 中日并排简介现在能完整截取中文段，不再丢末尾几个字
- 请求超时不再显示"暂时没法判断来源"的模糊提示

---

### 已知限制

- 未做代码签名 / 公证（macOS 首次打开需手动放行；macOS 不支持应用内静默更新）
- 仅发布 Windows x64 与 macOS arm64
