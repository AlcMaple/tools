搜索下载新增历史记录与下载源记忆、集数排除下载，媒体库扫描与切页优化，修复部分动漫下载失败

> [!Tip]
> 已装 v0.7.0 的用户：应用会自动检测并提示，点「重启安装」即可。

> [!Note]
> 本应用依赖系统 ffmpeg，请确保已安装并加入 PATH。
> Windows 从 [ffmpeg.org](https://ffmpeg.org/download.html) 下载；macOS 推荐 `brew install ffmpeg`。

> [!Tip]
> macOS 首次打开提示"无法验证开发者"时，请右键 App → 选择"打开"，
> 或执行 `xattr -d com.apple.quarantine /Applications/MapleTools.app`。

> [!Warning]
> 当前为开发中版本（v0.x），界面与数据格式可能在后续版本调整。

---

### 新增

**搜索下载**

- 搜索框新增历史记录，并自动记住上次选择的下载源，不用每次重新切换
- 下载时可在集数网格中排除指定集（如已下载过的），无需再拆成多段分别下载

**推荐**

- 推荐支持填写推荐人 / 推荐对方，接受推荐时可记录成功备注

**其他**

- 输入框与文本支持右键剪切 / 复制 / 粘贴 / 全选
- 侧边栏导航改为中文

---

### 优化

- 媒体库扫描移至后台并增量更新，大幅减少冷启动卡顿
- 页面切换优化

---

### 修复

- 修复部分动漫（如学园默示录）因下载链接处理不当导致的下载失败
- 修复多处界面在点击 / 切换时的轻微抖动
- 修复若干已知问题，提升稳定性

---

### 已知限制

- 未做代码签名 / 公证（macOS 首次打开需手动放行；macOS 不支持应用内静默更新）
- 仅发布 Windows x64 与 macOS arm64
