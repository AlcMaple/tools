# 替代 BGM —— 需要拿到的信息

> 以 app 界面实际显示的为准。★=必须，☆=可缺（缺了就降级/隐藏）。

## 0. 唯一标识
- ★ 每个条目一个**稳定不变的 ID**（当主键，全 app 靠它关联）

## 1. 搜索
- ★ 按**中文名/别名**搜，返回候选列表
- 每条候选：★标题 · ★封面 · ☆年份 · ☆评分 · ★ID

## 2. 条目详情
- ★ 封面
- ★ 中文名 + ☆日文原名
- ★ 简介
- ★ 类型（动画 / 漫画 / 小说）+ ☆子类型（TV / 剧场版 / OVA）
- ★ 话数 / 集数
- ☆ 放送日期
- ☆ 别名（多个）
- ☆ 评分（0–10）
- ☆ 题材标签（多个）
- ☆ 制作公司
- ☆ 主创（导演 / 监督 / 音乐 / 系列构成 / 人物设定 / 原作）
- ☆ 片长

## 3. 番剧周期表
- ★ 当季在播列表，**按播出星期分组**
- 每条：★标题 · ★封面 · ★播出星期 · ☆评分 · ★ID

## 4. 封面图
- ★ 国内能直连的封面图 URL
